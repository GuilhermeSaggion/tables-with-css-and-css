O que é HTML?

Tags de links

Função Comentário
Linkar páginas O “URL” é o link do site em que o usuário será direcionado. O texto entre as tags será o texto com o link a ser clicado.
Definir onde as páginas serão linkadas 1. Abre o link na mesma aba. É a configuração padrão, ou seja, se não usar nenhum atributo, o link vai abrir assim.
2. Abre o link em uma aba em branco.
Link para mandar um Email Ao clicar no link, o usuário será redirecionado para o corpo de um e-mail endereçado para o e-mail inserido.
Link para uma ligação Ao clicar no link, o usuário telefonará para o número inserido. É necessário colocar o código do país, e o número sem espaço.
Direcionar para uma parte da mesma página Sendo que “nome” é qualquer nome desejado por você. A primeira tag vem com “#” e o nome, na segunda é o mesmo nome apenas. Coloque a segunda tag na parte que você quer direcionar o usuário, pode ser em um parágrafo, título, ou qualquer outro elemento.

Tags de inputs

Função Comentário
Criar caixas para inserção de
informações		 Inputs podem ou não ficar dentro de formulários. Eles coletam dados.
Autocompletar informações Se a caixa for coletar um e=mail, substitui “tipo de texto” por e-mail. Se aplica com telefones, nomes, endereços, cpf, entre outros dados que em algum momento você já inseriu no navegador e ficaram armazenados.
Focar em um input Ao carregar a página, o cursor já vem no input. Só é possível colocar um por página. Se tiver mais de um, o navegador vai focar no que aparecer primeiro.
Desabilitar um input Deixa o input bloqueado, sem poder receber informações. É usado quando em sites que você precisa preencher inputs na ordem para destravar o próximo.
Já criar um input com uma informação Onde o texto é a informação pré-estabelecida.
Criar um input para apenas leitura A informação não pode ser alterada, é possível apenas ler
Criar um texto de fundo O texto escrito entre aspas preenche o input até que o usuário insira uma informação.
Vincular um nome ao input Digita o nome do input e depois coloca o input entre as tags. Fazendo dessa forma, o navegador vai associar o nome ao input, coisa que ele não faz se você digitar sem essa tag.
Vincular um input a um label Dessa forma, você vincula uma label com um input sem que seja necessário que o input esteja entre as tags da label. Basta que o nome entre aspas seja o mesmo para os dois.
Input para senhas Ao preencher o input, os dados não serão mostrados
Input para números Só aceita que sejam inseridos números no input.
Input para emails Ele não aceita a informação se não tiver um formato de e-mail. Ele não verifica se o e-mail é válido.
input para URL Só aceita a inserção de um URL. Não verifica se ele é válido.
Inserir um checkbox Insere um checkbox.
inserir um círculo de preencher Insere o circulo. Geralmente são usados em conjunto. É possível configurar para só poder selecionar um do conjunto.
Inserir arquivos Ao clicar no botão, o usuário será direcionado aos seus documentos para anexar o que for desejado.
Para anexar arquivos, a tag form tem que ter os atributos: method=”post” e enctype=”multipart/form-data”
Inserir arquivos de extensões
específicas		 1. O input só aceitará a extensão de arquivo informada.
2. O input só aceitará o arquivo se for do tipo e da extensão informados. No exemplo, ele só aceita imagem em png. Ao usar o “*”(asterisco) o input aceitará todos os formatos.
Inserir múltiplos arquivos Mais de um arquivo será anexado.
Inserir um input de cor 1. Inserir um input para selecionar e enviar uma cor.
2. Inserir um input para selecionar e enviar uma cor com uma cor inicial, podendo ser o nome em inglês ou o código RGB
Inserir barra de arrastar 1. Inserir barra de arrastar
2. Estipular valores máximos e mínimos da barra
3. Com os limites definidos, insere um valor inicial.
4. Determina de quanto em quanto os valores da barra vão aumentando a cada “passo”. Se não colocar nada, vai ser de 1 em 1.
Estabelecer quantidade mínima e máxima de caracteres Estabelece a quantidade mínima de caracteres para que seja possível enviar uma informação. No exemplo ao lado, essa quantidade é de 5 caracteres. Se não tiver ao menos 5 caracteres, na hora de enviar o formulário, o html vai barrar. O mesmo acontece com a quantidade máxima.
Aparecer o teclado virtual certo Essa tag define, em celulares e dispositivos de teclado virtual, qual teclado aparecerá, se é o de letras, números ou códigos.
Vincular um input a um label Dessa forma, você vincula uma label com um input sem que seja necessário que o input esteja entre as tags da label. Basta que o nome entre aspas seja o mesmo para os dois.
Input para senhas Ao preencher o input, os dados não serão mostrados
Input para números Só aceita que sejam inseridos números no input.
Input para emails Ele não aceita a informação se não tiver um formato de e-mail. Ele não verifica se o e-mail é válido.

Tags de tabelas

Função Comentário
Criar tabelas Abre e fecha uma tabela
Criar bordas 1.Coloca bordas na tabela, onde 2 pode ser qualquer valor numérico, pois define a grossura em pixels
2. E red pode ser qualquer cor para a borda.
Geralmente essa função é feita pelo CSS
Criar espaçamento entre células Cria o espaçamento das células, onde 10 pode ser qualquer valor para o espaçamento, dado em pixels.
Geralmente essa função é feita pelo CSS
Inserir linhas Cria as linhas da tabela.
Separar células do cabeçalho da tabela O que estiver entre as tags estará na mesma célula da tabela. É usado no cabeçalho da tabela, pois dá destaque ao texto.
Separar células do corpo da tabela O que estiver entre as tags estará na mesma célula da tabela. É usado no corpo da tabela, pois não destaca o texto.

Tags de imagens

Função Comentário
Inserir texto alternativo O texto inserido entre as aspas irá aparecer no lugar da imagem se ocorrer algum problema no carregamento da imagem no site. Além disso, programas de leitura de site para deficientes visuais vai ler o texto quando passar pela imagem.
Tamanho da imagem 1. Define a largura em pixels.
2. Define a altura em pixels.
Geralmente essa função é feita pelo CSS
Se você alterar apenas uma dimensão, a outra será alterada na mesma proporção
automaticamente.
Adicionar legenda à imagem Primeiro abre a tag figure, ai dentro dela coloca a imagem e todas suas propriedades, e em seguida abre a tag figcaption, e o texto que estiver entre essa tag será a legenda.

Tags de videos

Função Comentário
Adicionar vídeo Insere o vídeo, e a propriedade controls insere os botões e a minutagem do vídeo.
Tamanho do vídeo 1. Define a largura em pixels.
2. Define a altura em pixels.
Geralmente essa função é feita pelo CSS
Se você alterar apenas uma dimensão, a outra será alterada na mesma proporção automaticamente.
Configurações do vídeo 1. O vídeo já começa assim que carregar a página
2. Assim que o vídeo acabar, ele já inicia novamente
3. O vídeo começa mutado. Alguns navegadores bloqueiam o autoplay, logo o vídeo não começa sozinho, porém permitem que o vídeo comece se ele estiver mutado.
Thumbnail Insere uma imagem de capa para o vídeo, uma thumbnail.
Extensão do vídeo Essa tag vai entre as tags de vídeo. Você colocará o mesmo vídeo várias vezes,mas em diferentes extensões. Se o navegador não conseguir rodar a primeira extensão que você escolheu, ele tentará a segunda, e assim por diante. O formato universal dos vídeos é o mp4, pois a maioria dos navegadores consegue lê-lo. Caso nenhuma extensão funcione, você pode colocar um texto para aparecer no lugar do vídeo, basta escrevê-lo entre as tags de vídeo.

Tags de Iframes

Função Comentário
Colocar uma página dentro de
outra.		 Anexa uma telinha dentro da página com a página do link dentro. Alguns sites não permitem que sejam anexados. É possível alterar as dimensões da telinha. Da pra anexar vídeos do youtube ou mapas do google, por exemplo.

Tags de forumulários

Função Comentário
Criar um formulário para coletar dados Abre e fecha um formulário
Definir onde os dados do formulário serão enviados Os dados coletados serão enviados no URL da tag. Se não usar essa tag, os dados vão ser enviados para a mesma URL.
Definir o método que os dados serão
enviados		 1. Envia os dados por meio da URL. Dessa forma, os dados ficam visíveis >no link do navegador, por isso é utilizado quando os dados não são confidenciais.
2. Envia os dados por meio da URL, mas, dessa forma, os dados não ficam visíveis, por isso é usado quando os dados são confidenciais.

Tags de botões

Função Comentário
Botão para enviar as informações do input O texto que vai aparecer no botão deve ser inserido entre as tags. Só é permitido um desse botão por input/formulário.
Botão clássico Ao clicar no botão ele faz uma determinada ação.
Botão de reset Reseta todas as informações dos inputs do formulário.
Botão desabilitado Deixa o botão bloqueado, sem poder ser comutado. É usado em sites que informações precisam ser preenchidas antes que ele seja comutado.
Botão autofoco Foca automaticamente em algum botão. Só pode haver uma dessa propriedade em toda a página
